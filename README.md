# sao-esmodule-mold

> es module SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao esmodule-mold my-project
```

### From git

```bash
sao lpreterite/sao-esmodule-mold my-project
```
